# Thief Detection and Alert System

A surveillance system developed using computer vision techniques to detect unauthorized human presence in restricted areas.

## Features

- Motion detection in restricted areas
- Human presence detection using OpenCV
- Automated alert generation using SMS and Email
- Real-time monitoring support

## Tech Stack

- Python
- OpenCV
- Twilio

## How to Run

```bash
pip install opencv-python twilio
python detect.py
```

## Project Overview

This project uses computer vision techniques to monitor restricted environments and detect suspicious activity. When unauthorized movement is detected, the system automatically generates alerts and notifications.
